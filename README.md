# Chess with React

An interactive chess game built with React.

## Getting Started

Install dependencies:
```bash
npm install
```

Run the development server:
```bash
npm start
```

Open [http://localhost:3000](http://localhost:3000) to play.

## Features

- Interactive chess board
- Move validation
- Pawn promotion
- Move history with takeback
- Game end detection

## Build

```bash
npm run build
```
